***************************************************
How To: Install CS-Cart on a VPS/VDS with \*\nix OS
***************************************************

These articles deal with installing CS-Cart on a server that runs a Unix-like operating system. 

This requires some technical skills, so use these tutorials **only for demonstration purposes**.

Contact a technical specialist to set up a server for your actual store.  

.. note::
    You need `FTP or SSH access <http://kb.cs-cart.com/ftp-account>`_ to your server. Contact your hosting provider for more details.
