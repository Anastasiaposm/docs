*********
Marketing
*********

The **Marketing** section allows you to:

* create :doc:`promotions and discounts <../manage_products/promotions/index>`;

* manage :doc:`newsletters <../addons/newsletters/index>`;

* create :doc:`gift certificates <../addons/gift_certificates/index>`;

* work with :doc:`eBay templates <../addons/eBay/ebay_templates>`;

* manage :doc:`banners <../addons/banners_management/index>`;

* add :doc:`reward points for orders <../addons/reward_points/order_auto>`.

.. important::

    *Newsletters*, *Banners*, *Gift certificates*, *eBay templates* and *Reward points* won't appear in the **Marketing** section unless you activate the corresponding add-ons.
