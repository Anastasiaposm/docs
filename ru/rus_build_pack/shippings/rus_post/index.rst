Почта России
------------

Описание
========

Автоматический расчёт стоимости доставки Почтой РФ входит в "Русскую локализацию CS-Cart" и "Русскую локализацию Multi-Vendor".

Вам доступны:

*   Несколько способов автоматического расчёта.

    -   Официальный тарификатор Почты РФ

        Для расчёта используется "Автоматический тарификатор внутренних отправлений" с официального сайта Почты РФ.

    -   russianpostcalc.ru

        Используется API бесплатного сервиса расчёта стоимости и срока доставки почтовых отправлений по России.

    -   eDost.ru

        Калькулятор стоимости доставки для 20+ курьерских служб России.


    Просто выберите самый удобный для Вас.

*   Печать бланков Почты РФ.
