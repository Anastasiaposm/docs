******************
Обновление CS-Cart
******************

Существует два способа обновить CS-Cart:

================
Центр обновлений
================ 

Начиная с CS-Cart/Multi-Vendor версии 4.0.x, вы можете использовать **Центр обновлений** для обновления своего магазина до последней версии:

.. toctree::
    :maxdepth: 2
    :titlesonly:
    :glob:

    upgrade_process
    resolve_conflicts
    cscart_not_found

===============
Импорт магазина
=============== 

Вы можете использовать модуль **"Импорт магазина"** для обновления вашего магазина с версий 2.2.4/2.2.5 или 3.0.x до версии 4.x.x.

Для обновления необходимо создать отдельную папку с установленным CS-Cart/Multi-Vendor и с помощью модуля импортировать данные из старого магазина.
