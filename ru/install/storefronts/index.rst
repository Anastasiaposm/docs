***************************
Настройка нескольких витрин
***************************

В CS-Cart можно из одной панели администратора управлять несколькими витринами. Информация о всех витринах хранится в одной базе данных, но для них можно по-разному настроить внешний вид, ассортимент товаров и способы оформления заказа. Учетные записи пользователей могут быть общими для всех витрин, или же отдельными для каждой витрины. При необходимости можно сделать так, чтобы пользователи воспринимали разные витрины как разные магазины.
