Установка на компьютер c Windows
--------------------------------

Рассмотрим установку пробной версии на локальный компьютер под управлением Windows. 

Все, что вам нужно, это скачать среду PHP-разработки XAMPP для вашей операционной системы и запустить стандартную процедуру установки (Распаковать архив → Запустить установку в браузере). В других системах (Denwer) необходимо выполнить аналогичные действия. 

.. note::

    XAMPP является самой популярной средой PHP-разработки, содержащей HTTP-сервер Apache, базу данных MySQL и PHP-интерпретатор. 

После развертывания веб-сервера на локальном компьютере, установка CS-Cart займет у вас всего несколько минут.
