Расширение функционала ядра с помощью хуков
*******************************************

Хуки позволяют расширить существующий функционал или добавить новый, без вмешательства в ядро платформы.

.. important::

    Важно!

    Любую разработку выполнять с помощью модуля и хуков. Если нет подходящего хука, просто напишите нам.
