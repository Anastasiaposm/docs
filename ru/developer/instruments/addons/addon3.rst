Файл с переводами
=================

Переводы добавляются с помощью файла «.po», который должен быть расположен в папке:
/var/langs/en/addons/[id_модуля].po

.. important::

    Для каждого яыка потребуется отдельный «.po» файл с переводом. Размещать дополнительные переводы необходимо в папке с соответсвующим кодом языка:
    /var/langs/[код_языка]/addons/[id_модуля].po

    Значение ``msgid`` должно быть одинаковым для всех языков.
