*******************
Раскрывающийся блок
*******************

Инструкция о том, как сделать раскрывающийся по клику блок.

.. fancybox:: img/toggle_start.png
    :alt: Popup Ajax

Нажмите здесь для детальной информации
--------------------------------------

Всё очень просто! Достаточно:

1.  Создать новый div с уникальным id , например, ``id="my_div"``. 

2.  Создать новую ссылку (кнопку) и задать id как у блока, только с префиксом ``sw_``, в данном случае: ``id="sw_my_div"``.

3.  Добавить на ссылку класс ``cm-combination``.

Всё.

Иконки переключатели или «Вкл/Выкл»
-----------------------------------

Можно использовать иконки или кнопки «Вкл/Выкл», для этого используются id для ссылок:

*   ``on_[id_блока]`` — открыть

*   ``off_[id_блока]`` — закрыть

``[id_блока]`` — уникальный id блока который нужно открывать/закрывать

Пример. Два раскрывающихся блока. Один с помощью иконки, второй текстовыми ссылками. Также, возможно использовать изображения ``<img />``
