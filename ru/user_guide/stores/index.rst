****************
Магазины/Витрины
****************

В `CS-Cart Ultimate <https://www.cs-cart.ru/cs-cart-ultimate-rus-pack.html>`_ существует возможность управления несколькими витринами с помощью одной панели администратора. На разных витринах может отличаться ассортимент товаров, список категорий, набор настроек и процесс оформления заказа. Поэтому можно сделать так, что разные витрины будут восприниматься покупателями как отдельные магазины. Данный раздел документации описывает процесс работы с несколькими витринами.

Чтобы настроить мультивитринность на вашем веб-хостинге, :doc:`воспользуйтесь этой инструкцией. </install/storefronts/index>`
