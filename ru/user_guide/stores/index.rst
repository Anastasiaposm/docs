****************
Магазины/Витрины
****************

В CS-Cart существует возможность управления несколькими витринами с помощью одной панели администратора. На разных витринах может отличаться ассортимент товаров, список категорий, набор настроек и процесс оформления заказа.Поэтому можно сделать так, что разные витрины будут восприниматься покупателями как отдельные магазины. Данный раздел документации описывает процесс работы с несколькими витринами.

Чтобы настроить мультивитринность на вашем веб-хостинге, :doc:`воспользуйтесь этой инструкцией. <../../install/storefronts/index>`

.. note::

    Для управления несколькими витринами с помощью одной панели администратора необходимо `приобрести соответствующее количество лицензий на дополнительные витрины <https://www.cs-cart.ru/dopolnitelnaya-vitrina.html>`_.
