*********
Категории
*********

Каталог в CS-Cart строится из **товаров** и **категорий** (групп товаров). Система категорий позволяет организовать каталог в виде дерева со множеством веток, чтобы покупателям легче было найти нужный товар.

.. important::

    У каждого товара в каталоге должна быть хотя бы одна категория.

У каждой категории может быть неограниченное количество дочерних категорий (так называемых **подкатегорий**). У дочерних категорий могут быть свои дочерние категории. Категории без родительских категорий называются **корневыми**.
 
.. note::

    Подробности об использовании категорий в условиях мультивитринности будут :doc:`в отдельной статье </user_guide/stores/index>`.

========================
Действия над категориями
========================

Администраторы могут выполнять на странице **Товары → Категории** следующие действия:

------------------
Добавить категории
------------------

* Чтобы добавить одну категорию, нажмите кнопку **+** в правой верхней части страницы.

* Чтобы добавить сразу несколько категорий, нажмите на кнопку с изображением шестерёнки в правой верхней части страницы и выберите вариант **Добавить несколько категорий**.

-----------------------
Редактировать категории
-----------------------

* Чтобы изменить свойства одной определённой категории, нажмите на название категории. Или же, нажмите на кнопку с изображением шестерёнки (появляется при наведении на категорию) и выберите вариант **Редактировать**.

* Чтобы изменить свойства нескольких категорий одновременно, отметьте эти товары галочками и нажмите на кнопку с изображением шестерёнки в правой верхней части страницы. Выберите вариант **Редактировать выбранные**.

-----------------
Удалить категории
-----------------

* Чтобы удалить определённую категорию, нажмите на кнопку с изображением шестерёнки (появляется при наведении на категорию) и выберите вариант **Удалить**.

* Чтобы удалить сразу несколько категорий, отметьте их галочками, нажмите на кнопку с изображением шестерёнки в правой верхней части страницы и выберите вариант **Удалить выбранные**.

.. warning::

    Будьте осторожны при удалении категорий! В Multi-Vendor дочерние категории и товары удаляются вместе с родительской категорией. В CS-Cart товары из удалённой категории помещаются в :doc:`категорию-корзину <trash_category>`.
