**************
Характеристики
**************

**Характеристики** — это дополнительные *неотделимые* свойства товара. С помощью характеристик вы можете добавить информацию, которая связана со спецификой ваших товаров. Например, если вы продаёте бумажные книги, вы можете создать поле для ввода ISBN; если вы продаёте музыку, вы можете создать поле для указания формата файла.

На странице **Товары → Характеристики** находится список всех характеристик, которые доступны в вашем магазине. Если нужно, вы можете организовать их в группы, а затем редактировать уже свойства всей группы, а не каждой отдельной характеристики.

.. hint::

    Используйте меню справа, чтобы переключаться между характеристиками и группами характеристик.

.. note ::
