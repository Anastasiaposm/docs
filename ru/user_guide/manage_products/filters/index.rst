*******
Фильтры
*******

Фильтры отображаются на витрине и позволяют покупателям быстро находить товары с определёнными свойствами (диапазоном цен, производителем, и т.п.). Фильтровать товары можно как по стандартным свойствам, так и по дополнительным свойствам (характеристикам), которые создаются на странице **Товары → Характеристики**.

.. note::

    Если поменять статус характеристики на *Скрыто* или *Выкл.*, то фильтр по этой характеристике будет выключен автоматически.

Фильтры можно сделать доступными как для всего магазина, так и только для конкретных категорий.

.. important::

    Если фильтры товаров не появляются на витрине, создайте для них специальный блок на странице **Дизайн → Макеты**.
