*****************
Отчёты о продажах
*****************

На странице **Заказы → Отчеты о продажах** находится статистика продаж, сделанных в вашем интернет-магазине. Статистика может быть представлена в виде таблиц или диаграмм. Для удобства, таблицы и диаграммы группируются по отчётам. Вы можете использовать отчёты, которые уже есть по умолчанию, или создать новые.

.. hint::

    Переключайтесь между отчётами с помощью меню справа.
