********
Отгрузки
********

CS-Cart позволяет разбивать товары в заказе на разные партии и отправлять их покупателям по отдельности. Эти партии называются **отгрузками** и позволяют облегчить работу с заказами, где есть товары, которых ещё нет в наличии.

Например, вы можете отправить товары, которые есть в наличии, сразу после размещения заказа, а отсутствующие на складе товары отправить позже отдельной отгрузкой.

====================
Просмотреть отгрузки
====================

#. Чтобы увидеть все отгрузки в магазине, откройте станицу **Заказы → Отгрузки**.

   .. fancybox:: img/shipment_list.png
       :alt: Список существующих отгрузок находится на странице Заказы → Отгрузки.

#. Нажмите на ID любой отгрузки, чтобы открыть детальную страницу этой отгрузки. Там вы сможете отредактировать комментарий и дату. Не забудьте нажать кнопку **Сохранить изменения**.

   .. fancybox:: img/shipment_details.png
       :alt: Вы можете редактировать комментарий и дату отгрузки.
