******************
Запросы на возврат
******************

На странице **Заказы → Запросы на возврат** можно работать с запросами от покупателей на возврат товаров. Также здесь можно настроить политику возвратов в магазине.

.. note::

    Чтобы разрешить возвраты в магазине, включите модуль :doc:`"Возврат товаров" </user_guide/addons/rma/index>` на странице **Модули → Управление модулями**. Там же вы сможете задать общие настройки, связанные с возвратами.

Раздел состоит из 4 подразделов: **Запросы на возврат**, **Причины возврата**, **Варианты компенсации** и **Статусы запросов на возврат**. Чтобы переключаться между подразделами, используйте кнопку с изображением шестерёнки.
