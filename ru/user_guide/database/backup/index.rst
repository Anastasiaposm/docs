***************************
Резервные копии базы данных
***************************

Здесь описано, как создавать и восстанавливать резервные копии базы данных. Рекомендуем периодически делать резервные копии — так можно вернуться к состоянию магазина в определенный момент времени.

.. note::

    В базе данных хранится не вся информация из магазина. Например, картинки лежат в подпапках установленного CS-Cart/Multi-Vendor. Поэтому, если нужно сделать полную резервную копию, резервируйте не только базу данных. При создании резервной копии есть  возможность включить в нее и файлы.
