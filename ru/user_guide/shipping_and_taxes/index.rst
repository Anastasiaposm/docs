****************
Способы доставки
****************

Список способов доставки можно отредактировать на странице **Администрирование → Доставка и налоги → Способы доставки**. Способ доставки в CS-Cart — это набор правил, который позволяет автоматически рассчитывать стоимость доставки в зависимости от:

* адреса доставки;

* стоимости заказа;

* веса товаров в заказе;

* количества товаров в заказе;

* индивидуальных настроек каждого товара.

Покупателю будут предложены способы доставки, которые подходят для его заказа.
