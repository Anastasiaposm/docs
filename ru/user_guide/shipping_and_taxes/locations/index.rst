*************
Тарифные зоны
*************

На странице **Администрирование → Доставка и налоги → Тарифные зоны** можно редактировать места, в которые ваш магазин осуществляет доставку. *Тарифная зона* — это набор параметров, которые относятся к определённой географической области.

.. important::

    Если адрес покупателя не соответствует ни одной из существующих тарифных зон (или для зоны, где находится покупатель, не заданы тарифы у способов доставки), то покупатель не сможет выбрать способ доставки и оформить заказ.

.. hint::

    Тарифная зона **"Все страны"** существует по умолчанию и срабатывает для всех покупателей, которые не относятся ни к одной из существующих тарифных зон. Создайте способ доставки *"Обсудить с менеджером"* и сделайте его бесплатным для тарифной зоны **"Все страны"**. Так любой покупатель сможет оформить заказ, если только он не относится к тарифной зоне, для которой не заданы тарифы ни у одного из способов доставки.
