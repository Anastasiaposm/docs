*****************
Пункты назначения
*****************

На странице **Администрирование → Доставка и налоги → Пункты назначения** можно редактировать места, в которые ваш магазин осуществляет доставку. Пункт назначения — это набор параметров, которые относятся к определённой географической области.

.. important::

    Если адрес покупателя не соответствует ни одному из существующих пунктов назначения (или для пункта назначения покупателя у способов доставки не заданы тарифы), то покупатель не сможет выбрать способ доставки и оформить заказ.

.. hint::

    Пункт назначения **"Все страны"** существует по умолчанию и срабатывает для всех покупателей, которые не относятся ни к одному из существующих пунктов назначения. Создайте способ доставки *"Обсудить с менеджером"* и сделайте его бесплатным для пункта назначения **"Все страны"**. Так любой покупатель сможет оформить заказ, если только он не относится к пункту назначения, для которого не заданы тарифы ни у одного из способов доставки.
