*******************************
Местоположение продавцов [Beta]
*******************************

.. note::

    Этот модуль работает исключительно в Multi-Vendor и впервые появился в версии 4.8.1.
    
Модуль “Местоположение продавцов [Beta]” дает покупателям больше информации для принятия решения о покупке товаров. Для этих целей у модуля есть несколько инструментов:

#. Карта, на которой расположены все продавцы.

#. Фильтр, который позволяет отображать на карте только продавцов из определенного города.

#. Список ближайших продавцов.

#. Фильтр товаров по расстоянию до покупателя.

#. Возможность менять местоположение покупателя, чтобы по-другому отфильтровать товары и вывести список ближайших продавцов.
