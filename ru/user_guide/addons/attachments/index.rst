*********************
"Прикреплённые файлы"
*********************

Модуль "Прикреплённые файлы" позволяет прикреплять файлы к товарам из вашего магазина. Прикрепите любые файлы, которые могут быть полезны покупателю(например, схемы или инструкции для пользователей. Покупатели могут скачать прикрепленные файлы на вкладке **Прикреплённые товары** на странице товара.
