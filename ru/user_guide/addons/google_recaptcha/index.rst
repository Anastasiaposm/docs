****************
Google reCAPTCHA
****************

.. important::

    Начиная с версии CS-Cart 4.5.1, модуль Google reCAPTCHA — единственный способ защиты от ботов. Старый анти-бот был вырезан.

Модуль **Google reCAPTCHA** защищает магазин от ботов и при этом не усложняет навигацию по сайту для обычных пользователей.

Модуль поддерживает две версии reCAPTCHA: reCAPTCHA v2: Checkbox и reCAPTCHA v3:

* *reCAPTCHA v2: Checkbox* использует физическую проверку в виде тестов. Обычно пользователям достаточно поставить галочку в поле "Я не робот". Тест CAPTCHA нужно будет пройти только если полученных данных недостаточно, чтобы подтвердить, что пользователь — не бот.

* *reCAPTCHA v3* невидима для пользователей магазинов и осуществляет проверку и защиту от ботов в фоновом режиме. Поддержка *reCAPTCHA v3* добавлена, начиная с версии CS-Cart 4.12.1.
