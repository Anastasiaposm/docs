*****************
"Возврат товаров"
*****************

Модуль "Возврат товаров" позволяет упростить процесс управления возвратом товаров. Когда модуль активен, покупатели регистрируют запросы на возврат товаров, указывая причину возврата и любую дополнительную информацию. После этого администраторы магазинов обрабатывают запросы при помощи специального раздела в панели администрирования.

О том, как обрабатывать зарегистрированные запросы, читайте в статье о :doc:`запросах на возврат </user_guide/orders/return_requests/index>`. Запросы на возврат доступны лишь для тех товаров, которые отмечены, как подлежащие возврату. Эта опция так же, как и период возврата, настраивается отдельно для каждого товара.
