Модули или аддоны
-----------------

Модуль (аддон) в CS-Cart позволяет, **без вмешательства в код платформы**, изменить или добавить:

*   новые инструменты, возможности и функции в панели администратора и витрине интернет-магазина;

*   внешний вид (дизайн);

*   отобразить или изменить отображение информации, добавить новые блоки;

*   интегрировать (связать) платформу со сторонними сервисами и API;

Модуль можно включить или выключить в любой момент. Выключенный модуль не влияет на работу платформы.

Добавление новых функций с помощью модуля позволит избежать конфликтов при обновлении платформы.

Для платформы CS-Cart существует множество полезных модулей, ознакомится с ними вы можете на `Marketplace <http://marketplace.cs-cart.com>`_.

Список модулей
==============

.. list-table::
    :stub-columns: 1
    :widths: 5 30

    *   -   Модуль
        -   Краткое описание

    *   -   Google Analytics 
        -   Интегрирует сервис Google Analytics в магазин

    *   -   Searchanise
        -   Сервис быстрого поиска

    *   -   SEO
        -   Добавляет поддержку удобных для восприятия человеком URLs (ЧПУ)

    *   -   Бестселлеры и товары со скидкой
        -   Собирает информацию о наиболее покупаемых товарах и товарах со скидкой, а также позволяет заполнять блок товаров подобными товарами

    *   -   Импорт магазина
        -   Позволяет вам импортировать данные из любой версии, начиная с 2.2.4 (за исключением редакции Community), в текущую.

    *   -   Мои изменения
        -   Шаблон (пример) модуля для ваших изменений

    *   -   Отложенные товары
        -   Позволяет пользователям откладывать товары, добавляя их в специальный список

    *   -   Прайс-лист
        -   Генерирует настраиваемые прайс-листы в формате PDF и XLS

    *   -   Твигмо
        -   Подключает к сервису Твигмо, генерирующему мобильную версию магазина для покупателей и предоставляющему мобильное приложение для администратора

    *   -   Теги
        -   Позволяет создавать теги для товаров

    *   -   Управление баннерами
        -   Позволяет создавать текстовые и графические баннеры

    *   -   Конструктор форм
        -   Позволяет создавать настраиваемые формы (например, "Связаться с нами")

    *   -   Hybrid Auth
        -   Авторизация с помощью социальных сетей

    *   -   Janrain
        -   Авторизация с помощью социальных сетей

    *   -   MYOB
        -   Программа управления бизнесом

    *   -   Quickbooks
        -   Самая популярная бухгалтерская программа для малого бизнеса в США

    *   -   RSS Лента
        -   Генерирует RSS ленту для товаров

    *   -   SMS-уведомления
        -   Позволяет настраивать SMS-уведомления для администратора магазина

    *   -   Антимошенничество
        -   Добавляет конфигурируемую проверку безопасности заказа, используя сервис Maxmind, чтобы избежать мошенничества

    *   -   Бонусные баллы
        -   Позволяет покупателю зарабатывать бонусные баллы за покупку товаров для их дальнейшего использования

    *   -   Водяные знаки
        -   Позволяет добавлять водяные знаки к изображениям товаров и категорий

    *   -   Комбинации товаров
        -   Создание комплектов товаров, со скидкой при покупке комплекта.

    *   -   Новости и новостные рассылки 
        -   Шаблон (пример) модуля для ваших изменений

    *   -   Обязательные товары
        -   Позволяет определять товары, которые необходимо купить для приобретения данного

    *   -   Ограничение доступа 
        -   Позволяет ограничить доступ к витрине магазина и администраторской части с разных IP-адресов с применением различных настроек

    *   -   Ограничение по возрасту
        -   Позволяет устанавливать возрастное ограничение на товары и категории

    *   -   Опросы и голосования
        -   Позволяет создавать настраиваемые опросы и голосования

    *   -   Отзывы и обсуждения
        -   Позволяет покупателям оставлять комментарии к товарам, категориям, заказам и т.д., а также вести обсуждения товаров и писать отзывы
	
    *   -   Подарочные сертификаты
        -   Подключает встроенную систему подарочных сертификатов

    *   -   Поддержка HiDPI
        -   Поддержка экранов Retina

    *   -   Поиск ближайших магазинов
        -   Позволяет отмечать магазины на карте и сообщать покупателям о ближайших к ним магазинах

    *   -   Поставщики
        -   Позволяет назначать поставщиков товарам

    *   -   Привилегии продавцов
        -   Позволяет решать, что продавцы могут и не могут делать на маркетплейсе

    *   -   Прикрепленные файлы
        -   Позволяет прикрепить файлы к товарам

    *   -   Режим каталога
        -   Позволяет использовать магазин как каталог товаров

    *   -   Сопутствующие товары
        -   Позволяет создать блок товаров, которые чаще всего покупают вместе с данным

    *   -   Социальные кнопки
        -   Добавляет кнопки популярных соц. сервисов (Twitter, Facebook и т.д.)
		
    *   -   Тарифные планы для продавцов
        -   Позволяет создавать тарифные планы для своих продавцов с различными условиями и ценами

    *   -   Увеличенные изображения товаров
        -   Показывает увеличенные изображения при наведении указателя мыши

    *   -   Штрихкоды
        -   Позволяет добавлять и настраивать штрих-коды к заказам

    *   -   Экспорт в Google
        -   Расширяет возможности модуля Экспорт данных для работы с сервисом Google Base

    *   -   Экспорт данных
        -   Позволяет экспортировать товары в CSV файлы. Возможен автоматический экспорт по расписанию
