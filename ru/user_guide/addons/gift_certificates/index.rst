**********************
Подарочные сертификаты
**********************

Модуль позволяет покупать и использовать в вашем магазине подарочные сертификаты. Они удобны для тех покупателей, которые не знают, какой подарок выбрать. Вместо этого они могут заказать в магазине сертификат со следующими параметрами:

* Имя и адрес (или электронная почта) человека, который получит сертификат.

* Стоимость сертификата в валюте магазина.

* Сообщение для получателя сертификата.

* Любые дополнительные товары, привязанные к сертификату; получателю эти товары достанутся бесплатно.

Когда подарочный сертификат создан, покупатель добавляет его в корзину и покупает как обычный товар. Администратор может настроить магазин так, чтобы сертификат активировался и отправлялся получателю только тогда, когда заказ получает определённый статус.

Получатель сертификата может посетить магазин, выбрать товары из каталога и добавить их в корзину. При оформлении заказа можно использовать сертификат — для этого нужно ввести код в соответствующее поле. Стоимость сертификата будет вычтена из суммы заказа. Если сертификат не полностью покрывает заказ, то остаток суммы можно оплатить любым доступным способом оплаты.

У сертификатов нет срока действия, и ими можно воспользоваться в любое время. Также можно использовать один сертификат в нескольких заказах, если стоимость сертификата не была полностью истрачена.

.. note::

    Единственное ограничение — нельзя подарочным сертификатом оплатить заказ, который содержит только ещё один подарочный сертификат.
