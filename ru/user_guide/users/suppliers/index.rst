*********
Поставщики
*********

**Поставщики** используются в базе данных, чтобы отметить товары, которые поставляются сторонними организациями. 

.. note::

    По умолчанию поставщики недоступны. Сначала нужно включить модуль **Поставщики** на странице **Модули → Управление модулями**.
