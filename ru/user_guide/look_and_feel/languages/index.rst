*****
Языки
*****

==================
Управление языками
==================

Вы можете управлять языками доступными в вашем магазине из **Администрирование → Языки → Языки**. На данной странице представлены две вкладки:

* **Установленные** — языки, установленные в вашем магазине.

* **Доступные** — языки, которые можно установить.

Значок **шестеренки** на панели языка позволяет редактировать, дублировать или экспортировать язык, а также обновлять для него переводы.

.. image:: img/installed_languages.png
    :align: center
    :alt: Вкладка с установленными языками

===============
Установка языка
===============

1. В панели администратора откройте **Администрирование → Языки → Языки**.

2. Переключитесь на вкладку **Доступные**.

.. image:: img/available_languages.png
    :align: center
    :alt: Вкладка с доступными языками

3. Выберите нужный язык и щёлкните по кнопке **Установить** рядом с ним.

=========================================
Язык интерфейса и язык элементов магазина
=========================================

В CS-Cart язык интерфейса и язык элементов страницы (например, название категории, описание товара и т.д.) могут различаться.

* Для изменения языка интерфейса панели администратора:

1. В правом верхнем углу найдите иконку с кодом языка.

2. Щёлкните по ней и выберите нужный язык.

* Для изменения языка элемента страницы:

1. В панели администратора откройте страницу редактирования нужного объекта (например, категория, товар и т.д.).

2. Щёлкните по названию языка в поле **Язык** в левом верхнем углу зоны редактирования объекта.

3. Выберите нужный язык.
