************************************************************************************
В режиме Виджета тема "Responsive" некорректно отображается на мобильных устройствах
************************************************************************************

Если при работе с темой **Responsive** вы перевели её в режим **Виджета** и встроили свой магазин в другой сайт, и при этом на мобильных устройствах ваш магазин растягивается по ширине версии для компьютера, добавьте следующий метатег на страницу, где встроен ваш магазин:

.. code-block:: none

    <meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=0">

Поместите его между тегами *<head>* и *</head>*, тема **Responsive** начнёт отображатсья корректно на всех мобильных устройствах.
