*****
Блоки
*****

Одновременно с контейнерами, формирующими структуру витрины, в макетах используются блоки, служащие для наполнения этой структуры содержимым и рабочими элементами. Блоки можно сравнить с коробками, в которых лежат отдельные компоненты витрины. Компонентами могут быть поисковые формы, переключатели языков и валют, меню, фильтры товаров, баннеры, списки товаров, текст, HTML-вставки и другое содержимое. В конечном итоге, все элементы витрины отображаются с помощью блоков.

Основное отличие блоков друг от друга — это тип данных, заложенных в них. Однако, даже блоки с одинаковым типом данных могут иметь различные наполнение, вид и настройки. Кроме того, не которые блоки применяются только в определённых случаях, например, для отображения определенного содержимого или только на отдельных страницах. Таким образом, даже несмотря на типичную структуру, с помощью блоков можно создавать разнообразные страницы с уникальным содержимым.

Изначально CS-Cart уже включает в себя несколько настроенных блоков. Их можно использовать для того, чтобы быстро добавить на витрину самые популярные и необходимые элементы. Также существует возможность создания индивидуальных блоков на основе доступных шаблонов.
