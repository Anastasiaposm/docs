********
Страницы
********

В панели администратора на странице **Веб-сайт → Страницы** можно создавать и публиковать следующие типы контента:

* **Страница** — содержит текст, отформатированный в HTML. На странице можно разместить информацию о компании, политике конфиденциальности и т.п.

* **Форма** (если включен модуль :doc:`"Конструктор форм" </user_guide/addons/form_builder/index>`) — содержит форму обратной связи, а иногда еще и текст, отформатированный в HTML. Знания HTML и другие технические навыки для создания формы в CS-Cart или Multi-Vendor не нужны — есть простой и удобный редактор.

* **Опрос** (если включен модуль :doc:`"Опросы и голосования" </user_guide/addons/polls/index>`) — опрос или голосование.

* **Ссылка** — ссылка на внешнюю веб-страницу или на одну из страниц магазина. Нужна для двух целей:

  * появляться среди других страниц с контентом :doc:`в созданном вами блоке </user_guide/look_and_feel/layouts/blocks/actions_on_blocks>` с типом "Страницы";

  * создать ссылку, которая внешне ведёт на ваш сайт, но на самом деле ссылается на другой ресурс, например, страницу в социальной сети.

    Вместо того, чтобы давать покупателям ссылку на ``https://www.facebook.com/example.company/``, вы сможете дать им ссылку ``https://example.com/facebook``, которая будет вести на ту же страницу.

    .. note::

        Чтобы так использовать ссылки, в магазине должен быть включен модуль :doc:`SEO </user_guide/addons/seo/index>` Иначе ссылка будет выглядеть так: ``http://example.com/?dispatch=pages.view&page_id=%pid%``.
