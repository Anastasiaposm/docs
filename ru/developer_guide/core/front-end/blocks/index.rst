*****
Блоки
*****

Функционал и поведение блоков регулируется схемами, расположенными в каталоге *app/schemas/block_manager*:

* **blocks.php** — главная схема, описывающая типы блоков;

* **blocks.functions.php** — функции, используемые главной схемой;

* **block_cache_properties.php** — параметры кеширования, применяемые глобально ко всем блокам, поддерживающим кеширование;

* **dispatch_descriptions.php** — список диспатчей и соответствующих языковых переменных. Нужна только для генерации списка для поля ``Dispatch`` при добавлении/редактировании локации;

* **templates.php** — описание параметров, которые доступны для Smarty-шаблонов, доступных для использования в блоках;

* **fillings.php** — описание заполнений (fillings);

* **dynamic_objects.php** — описание динамических объектов.
