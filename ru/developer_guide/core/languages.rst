**************************
Переводы и мультиязычность
**************************

Два основных механизма для переводов:

* :doc:`Сущности <lang_entities>` — перевод любых динамических данных (продукты, категории, страницы и т.д.).
    
* :doc:`Языковые переменные <language_variables>` — перевод текста самой платформы CS-Cart.

.. note: 

    Настройки (settings) также являются сущностями. Их перевод осуществляется так, же как и перевод других сущностей. Однако интерфейса для перевода настроек нет. Внешне настройки наоборот можно переводить через сервис переводов наравне с языковыми переменными.

====================
Сервис для переводов
====================

На текущий момент мы используем сервис `Crowdin <https://crowdin.com/>`_. Перевод CS-Cart осуществляется в рамках проекта `cs-cart-latest <https://crowdin.com/project/cs-cart-latest>`_.
