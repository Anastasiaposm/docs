Почта России
------------

Описание
========

Автоматический расчёт стоимости доставки Почтой РФ входит в "Русскую локализацию CS-Cart".

Вам доступны:

*   Несколько способов автоматического расчёта.

    -   Почта России (pochta.ru)

        Расчет выполняется с сайта: |link|

        .. |link| raw:: html

           <!--noindex--><a href="https://www.pochta.ru" target="_blank" rel="nofollow">Почта России</a><!--/noindex-->.

    -   russianpostcalc.ru

        Используется API бесплатного сервиса расчёта стоимости и срока доставки почтовых отправлений по России.

    -   eDost.ru

        Калькулятор стоимости доставки для 20+ курьерских служб России.


    Просто выберите самый удобный для Вас.

*   Печать бланков Почты РФ.
